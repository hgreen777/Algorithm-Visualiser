# Algorithm-Visualiser
Visualise Sort &amp; Search Algorithms. [Python]

# 🧾 Features
- Select an algorithm
- Play/Stop an algorithm
- Step through an algorithm
- Show the current state of the array [algorithm]
- Show the cursor (pointer) position
